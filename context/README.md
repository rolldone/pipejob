# Windows Test Context for pipejob

This folder contains Windows-focused test/checklist documents to validate the `idle_timeout` and process-tree termination behavior added to `sub_app/pipejob`.

Files included:
- `test-plan.md` — test cases and acceptance criteria
- `test-environment.md` — setup steps and preconditions for a Windows test runner
- `test-scripts.md` — suggested commands/scripts to run and what to observe
- `bug-report-template.md` — template to capture failures and logs

Use these documents when running manual or CI-powered Windows verification runs.
